GUI Object has 3 type: 'window', 'button', 'menu'.
If you want to create root node, write.

'create,id,window,stating x position,starting y position,width,height,title'
'create,id,button,stating x position,starting y position,width,height,button text'
'create,id,menu,stating x position,starting y position,width,height,the number of items,menu name'

If you want to insert the node, write.

'insert,id,window,parents of node,stating x position,starting y position,width,height,title'
'insert,id,button,parents of node,stating x position,starting y position,width,height,button text'
'insert,id,menu,parents of node,stating x position,starting y position,width,height,the number of items,menu name'

If button's text is 'play' or 'pause', you can play or pause the music in the assignment file.

The browser dose not support canvas
